

1. Переместиться в папку Chat/src и загрузить её содержимое
2. Server.java - отвечает за работу сервера => сначала запускаем его; Предварительно смотрим и запоминаем IP адрес машины, на которой запустили сервер
3. Test.java - отвечает за работу любого из клиентов => запускаем клиентов на машинах одной сети
4. Открывается окошечко GUI клиента. Выбираем себе имя, вводим IP сервера(которое запомнили ранее), порт УКАЗЫВАЕМ 30009 !!!!! => жмём submit
5. Если всё хорошо и нет никаких непредвиденных проблем(ни разу не наблюдалось), то "you are connected to the server"
6. Можно для начала потестить клиента в режиме эха: ввести в качестве собеседника себя(своё имя) и посмотреть на чатик с самим собой
7. Потом можно попробовать запустить клиента ещё на одной машине и посмотреть как работает там
